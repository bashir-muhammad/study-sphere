Create a full design system for the following project:

## Project Summary

**Core Features:**

- **Home Page** - Browse and manage your study decks with quick access to create new decks and view your dashboard

- **Deck Creation** - Manually create flashcards with questions, answers, and multiple-choice options that are shuffled and validated

- **Study Mode** - Interactive learning with flip cards that rotate to reveal answers, and four multiple-choice options with immediate feedback

- **Results Page** - See your performance immediately after completing a study session with detailed statistics

- **Dashboard** - Comprehensive analytics showing overall progress, average scores, study sessions completed, and deck-by-deck statistics with charts

**Technical Highlights:**

- localStorage-based persistence for decks and study sessions (no auth required, all data stays in browser)

- Modern UI with gradient headers, smooth card animations, and a cohesive color system (indigo primary, amber accent)

- Fully responsive design that works beautifully on mobile, tablet, and desktop

- Type-safe TypeScript implementation with clean component architecture

- Recharts integration for visualizing performance data

**Key Pages:**

- `/` - Home with deck list

- `/create` - Build new decks

- `/learn/[id]` - Study with flip cards and multiple choice

- `/results` - Session results

- `/dashboard` - Performance analytics
